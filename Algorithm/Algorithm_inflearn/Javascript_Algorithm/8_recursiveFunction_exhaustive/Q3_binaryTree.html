깊이 우선 탐색(DFS): 루트 노드(or 다른 임의의 노드)에서 시작해서 다음 분기(branch)로 넘어가기 전에 해당 분기를 완벽하게 탐색하는 방법 예) 미로를 탐색할 때 한 방향으로 갈 수 있을 때까지 계속 가다가 더 이상 갈 수 없게 되면 다시 가장 가까운 갈림길로 돌아와서 이곳으로부터 다른 방향으로 다시 탐색을 진행 -> 모든 노드를 방문 하고자 하는 경우에 이 방법을 선택 전위순회: root를 먼저 방문 중위 순회: 왼쪽 하위 트리를 방문 후 root 방문 후위 순회: 하위 트리 모두 방문 후 root 방문 층별 순회: 위 쪽 node들 부터 아래방향으로 차례로 방문
